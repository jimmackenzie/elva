Elva

A calm, dark, Atom syntax theme for folks who love to ᕕ( ᐛ )ᕗ.

To install:

  1. Go to Preferences => Install
  2. Toggle "Themes" and search for "Elva"
  3. Press "Install"
  4. Head to Preferences => Themes and pick
    "Elva" from the dropdown as your syntax theme

Or: 

apm install elva-syntax
